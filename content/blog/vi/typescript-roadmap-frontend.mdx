# Lo trinh TypeScript cho frontend developer

TypeScript phat huy tot nhat khi hoc theo lo trinh va co bai tap thuc te.

## Tuan 1-2: Nen tang

- Kieu du lieu co ban.
- Union, interface, type alias.
- Function type va object type.

## Tuan 3-4: Nang cao

- Generic.
- Utility types.
- Narrowing va type guard.

## Tuan 5-6: Ap dung du an

- Typed API response.
- Typed form state.
- Tach model va UI contract ro rang.

## Ket luan

Sau 6 tuan, ban se viet code tu tin hon va giam loi runtime ro ret.
